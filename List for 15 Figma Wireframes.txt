Smart Student Wallet: Figma Subtasks

1. Welcome Screen Wireframe
   - Purpose: Initial splash screen with app name, logo placeholder, and “Get Started” button to proceed to login or signup.
2. Loading Screen Wireframe
   - Purpose: Brief loading animation placeholder (e.g., spinning icon) transitioning to welcome screen or login.
3. Login Wireframe
   - Purpose: Login screen with email/password fields, login button, forgot password link, and back to home link.
4. Signup Wireframe
   - Purpose: Signup screen with email/password fields, signup button, email verification prompt, and link to login page.
5. Dashboard Wireframe
   - Purpose: Dashboard showing mock balance (e.g., $100), pie chart placeholder for spending by category, and total expense summary.
6. Add Expense Wireframe
   - Purpose: Screen for adding expenses with amount, category dropdown (Food/Transport/Books/Entertainment/Other), date picker, and submit button.
7. Set Budget Wireframe
   - Purpose: Screen for setting a monthly budget with input field, save button, and notification placeholder at 80% usage.
8. Transaction History Wireframe
   - Purpose: Screen for viewing transaction history with list view, date/category filters, and export PDF button.
9. Forgot Password Wireframe
   - Purpose: Password reset flow with email input, send reset link button, and back to login link.
10. Budget Notification Wireframe
   - Purpose: Screen displaying 80% budget notification with options to view details or dismiss.
11. Expense Details Wireframe
   - Purpose: Detailed view of a single expense when tapped from history, showing amount, category, and date.
12. Profile Settings Wireframe
   - Purpose: Screen for managing account details (e.g., update email, change password) with a save button.
13. Money Transfer Wireframe
   - Purpose: Screen for transferring money with recipient email, amount field, transfer button, and confirmation prompt (meets money transfer requirement).
14. Help/Support Wireframe
   - Purpose: Help page with FAQs, contact support button, and back to dashboard link.
15. Export Confirmation Wireframe
   - Purpose: Screen confirming PDF export with download button and cancel option.
